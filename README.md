# TopEnergy SGS

## Instalação

1. Clone o repositório
2. Execute `npm install`
3. Configure o arquivo `.env` com suas credenciais
4. Execute o script `database.sql` no seu banco Supabase
5. Inicie o servidor: `npm start`

## Funcionalidades

- Simulador de economia solar
- Cadastro e login de usuários
- Dashboard com saldo de CSGS
- Painel administrativo
- Integração com blockchain (CSGS token)

## Tecnologias

- Node.js + Express
- PostgreSQL (Supabase)
- Web3.js (Ethereum)
- JWT para autenticação
